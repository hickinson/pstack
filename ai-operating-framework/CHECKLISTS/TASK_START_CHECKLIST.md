# Task Start Checklist

Before starting, check:

- Do I know what the task is?
- Do I know what a good result looks like?
- Do I know which overlay applies?
- Do I know what inputs I have?
- Do I know the main constraints?
- Do I know whether this needs review or approval?
- Do I know what evidence would count as done?

If two or more answers are no, run the Clarifier first.
