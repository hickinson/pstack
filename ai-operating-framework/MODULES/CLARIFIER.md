# Clarifier

## Purpose

Turn a rough ask into a task brief that is clear enough to act on.

## Inputs

- raw request
- context
- constraints
- desired outcome if known

## Outputs

- objective
- constraints
- assumptions
- inputs
- expected outputs
- success criteria
- open questions

## Operating rules

- reduce ambiguity
- make assumptions explicit
- identify what is missing
- avoid solving before defining

## Standard prompt pattern

Turn the following rough ask into a structured task brief.

Return:
- objective
- context
- constraints
- assumptions
- desired output
- success criteria
- risks or unknowns
