# Idea Processor

## Purpose

To turn a rough idea, note, or opportunity into a usable task starting point.

---

## Use when

- the starting point is a messy idea rather than a defined task
- you have a concept but not yet a workable problem statement
- you want to package raw thinking into something the rest of pstack can use

---

## Inputs

- raw idea, note, or opportunity
- any known context
- any obvious constraints or ambitions

---

## Outputs

- what the idea is really about
- the core problem or opportunity
- who it is for
- what a useful first version would look like
- a simple task definition suitable for clarification or briefing

---

## Operating rules

- reduce noise without removing useful intent
- prioritise usefulness over completeness
- do not expand scope unnecessarily
- make the proposed first version concrete
- keep the output suitable for handoff into the Clarifier or Task Brief

---

## Default invocation template

Take the following idea and turn it into a usable task starting point.

Return:
- what this idea is really about
- the core problem or opportunity
- who it is for
- what a useful first version would look like
- a simple task definition suitable for further refinement

Idea:
[insert raw idea]

Context:
[insert context if available]
